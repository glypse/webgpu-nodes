# webgpu-nodes

A web-based node-based fragment shader editor for wgsl

If you're reading this, this project is in its earliest of early stages - there's probably nothing to see here for now. Check back again later, maybe it'll be in alpha :)

## Technologies

- Svelte 5
- Svelte Flow
- Motion GPU

Thank you very much to all contributors of all technologies used in this project, see [apps/web/package.json](apps/web/package.json)
